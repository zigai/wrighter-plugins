Plugins for [Wrighter](https://github.com/zigai/wrighter)


- Resource blocker
- Network logger
- JSON scraper
- Stealth
- more...


# Installation
#### From PyPi
```
pip install wrighter-plugins
```
#### From source
```
pip install git+https://github.com/zigai/wrighter-plugins.git
```
